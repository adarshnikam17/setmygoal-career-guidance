SetMyGoal is a personalized career guidance platform designed for students across
different education levels (school to graduates).

## Vision
To guide every student based on their current education stage, interests,
and future goals instead of generic career advice.

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MongoDB

## Core Concept
Content and guidance dynamically adapt based on:
- Student class / education level
- Board / stream
- Interests
